Auto-hide Sidebar Demo

Features Implemented:

  • Auto-collapsed sidebar - Shows only icons by default (70px width)
  • Hover to expand - Sidebar expands to 260px on hover
  • Pin functionality - Click the pin button to keep sidebar expanded
  • Smooth transitions - All animations are smooth with 0.3s transitions
  • Tooltips - Hover over icons in collapsed state to see labels
  • LocalStorage persistence - Pin state is saved and restored
  • Responsive design - Works on all screen sizes

How to Use:

  1. The sidebar starts in collapsed mode showing only icons
  2. Hover over the sidebar to expand it and see full labels
  3. Click the pin button (ðŸ“Œ) to keep the sidebar expanded
  4. Click the pin button again to unpin and return to auto-hide mode
  5. In collapsed mode, hover over any icon to see a tooltip with the label
